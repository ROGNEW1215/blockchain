# 基于区块链的工业设备预测性维护系统

本项目是一个基于区块链技术的工业设备预测性维护系统，旨在提高工业设备的维护效率，降低故障风险，延长设备使用寿命。

## 核心功能模块

### 设备数据管理模块
- 数据采集与上传：支持通过物联网设备自动采集工业设备运行数据，也可手动上传设备历史数据
- 数据查询与展示：提供多维度查询功能，以表格、图表等形式展示设备运行数据
- 数据异常预警：实时监测数据，自动触发预警通知

### 故障预测模块
- 模型管理：管理故障预测模型，支持模型上传、更新与删除
- 故障预测：基于设备数据进行分析，生成故障预测报告
- 历史预测记录：保存和查询历史预测记录

### 维护管理模块
- 维护任务发布：发布维护任务，任务信息上传区块链
- 任务承接与执行：维护人员承接任务，记录执行进度
- 任务验收与结算：验收任务，基于智能合约自动结算

### 区块链信息模块
- 节点管理：管理区块链网络节点
- 智能合约管理：管理系统中的智能合约
- 区块信息查询：查询区块详情和交易记录

## 技术栈
- 前端：React.js, Ant Design
- 后端：Node.js, Express
- 区块链：Ethereum, Solidity
- 数据库：MongoDB
- 其他：Web3.js, TensorFlow.js (用于预测模型)

## 安装与运行
```bash
# 安装依赖
npm install

# 运行开发服务器
npm run dev

# 构建生产版本
npm run build
```
